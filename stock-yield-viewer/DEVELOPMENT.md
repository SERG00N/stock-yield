## 🐛 Известные проблемы

1. **MOEX API ограничения:**
   - Не все облигации имеют `COUPONVALUE`
   - Дата следующего купона может отсутствовать
   - История дивидендов не для всех акций (SBER)

2. **Vite:**
   - Требуется `--force` при изменении компонентов
   - Горячая перезагрузка иногда не срабатывает

3. **localStorage:**
   - Ограничение ~5-10MB
   - Данные теряются при очистке кэша
