# Прогресс разработки приложения "Мой инвестиционный портфель"

**Последнее обновление:** 2026-02-22

## Описание проекта

Веб-приложение для просмотра доходности акций и облигаций Московской биржи с функцией управления инвестиционным портфелем, отслеживанием купонных выплат и дивидендов.

**Технологический стек:**
- React 19.2, Vite 7, React Router DOM 7
- UI: Bootstrap 5, React Bootstrap, Bootstrap Icons
- Данные: MOEX ISS API (бесплатный, без ключей)
- Хранение: localStorage
- Тема: тёмная зелёная

## 🐛 Известные проблемы

1. **MOEX API не возвращает `COUPONVALUE` для всех облигаций**
   - Решение: ручной ввод купонов уже реализован

2. **MOEX API не возвращает историю дивидендов для всех акций (например, SBER)**
   - Решение: ручной ввод дивидендов уже реализован

3. **Требуется полная перезагрузка Vite (`--force`) при изменении компонентов**
   - Workaround: `npm run dev -- --force`

4. **Дата следующего купона может отсутствовать**
   - Решение: показывать "—", возможность ручного отслеживания

## 📝 Заметки для продолжения

### Ключевые функции API (`src/api/moex.js`)
```javascript
fetchStocks()           // Список акций
fetchBonds()            // Список облигаций
fetchCouponHistory(id)  // История купонов облигации
fetchDividendHistory(id) // История дивидендов акции
fetchNextCouponDate(id) // Дата следующего купона
fetchCandles(id, from, to) // Свечи для графика
```

### Хук портфеля (`src/hooks/usePortfolio.js`)
```javascript
portfolio          // Массив позиций
couponHistory      // История полученных купонов
dividendHistory    // История полученных дивидендов
addPosition()      // Добавить позицию
removePosition()   // Удалить позицию
confirmCoupon()    // Подтвердить получение купона
confirmDividend()  // Подтвердить получение дивиденда
```

### Структура позиции портфеля
```javascript
{
  id: number,
  securityId: string,    // Тикер бумаги
  ticker: string,
  name: string,
  type: 'stock' | 'bond',
  quantity: number,
  avgPrice: number,      // Средняя цена покупки
  purchaseDate: string,  // Дата покупки (ISO)
  receivedCoupons: number,
  totalCouponAmount: number,
  manualCoupons: [{date, amountPerBond}],
  receivedDividends: number,
  totalDividendAmount: number
}
```
